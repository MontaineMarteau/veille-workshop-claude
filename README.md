# Atelier 25 min, créez votre agent de veille automatique avec Claude Cowork

Cet atelier fait partie du cycle **Les Midis IA** de Peopulse.

À la fin de la session, vous repartez avec un agent de veille personnalisé qui s'exécute tout seul dans votre projet, à la fréquence de votre choix, et qui vous livre chaque semaine un rapport ciblé sur ce qui vous intéresse vraiment.

## Comment participer

1. Ouvrez Claude Cowork dans votre projet.
2. Collez ce prompt dans le chat,

   ```text
   Suis ce guide : https://raw.githubusercontent.com/MontaineMarteau/veille-workshop-claude/main/veille_workshop_guide.md
   ```

3. Laissez-vous guider. Cowork mène un entretien avec vous, crée votre skill, le planifie, lance un premier rapport de test.

## Programme (25 minutes)

| Temps       | Étape                                              |
| ----------- | -------------------------------------------------- |
| 0:00 - 0:05 | Lancement, vous ouvrez Cowork et collez le prompt  |
| 0:05 - 0:10 | Cowork mène l'entretien et génère votre skill      |
| 0:10 - 0:15 | Planification automatique via `/schedule`          |
| 0:15 - 0:20 | Premier rapport de test                            |
| 0:20 - 0:25 | Astuces et améliorations                           |

## Comment vous préparer

Rien à préparer. Venez avec votre projet ouvert dans Cowork.

## Ce que Claude Cowork va créer avec vous

- Un **skill de veille** personnalisé dans votre projet (`.agents/skills/domain-veille/SKILL.md`).
- Une **tâche planifiée** qui exécute ce skill à la fréquence de votre choix (par défaut, lundi 9h).
- Un **premier rapport de test** dans le dossier `veille_reports/`, généré pendant l'atelier.
- À chaque exécution future, le skill vous proposera dans le chat des **améliorations à lui apporter**, pour affiner votre veille au fil des semaines.
